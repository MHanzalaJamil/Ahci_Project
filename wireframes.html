Design Decisions — Wireframe Stage

Navigation model: Persistent 5-tab bottom nav chosen over hamburger menu — students need zero-friction access to core features in a hurry.

Map first: Map is tab 2 (not 3+) because room-finding is the primary use case per our survey data.

Theme toggle: Placed top-right of every page header — consistent, never occludes content, reachable with one thumb.

Alert severity: Left border colour + section grouping chosen over icon-only to serve colour-blind users.

Places hub: Single page with filter chips preferred over separate pages to reduce navigation depth.

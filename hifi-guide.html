Figma Implementation Note
This document defines the design tokens used in the CampusNav high-fidelity prototype. In Figma, create a Local Variables collection named CampusNav/Tokens and map each token below. Use Auto Layout for all components and Variants for dark/light theme switching. Frame size: 390 × 844 (iPhone 14 equivalent).
